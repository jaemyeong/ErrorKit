# ErrorKit

범용 오류 타입 모음.

## Requirements

- Xcode 13.2.1
- Swift 5.5

## Installation

### Swift Package Manager

```swift
.package(url: "https://github.com/jaemyeong/ErrorKit.git", .upToNextMajor(from: "0.1.3"))
```

## Usage

```swift
import ErrorKit

throw UnknownError()
```
