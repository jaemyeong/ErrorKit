# ErrorKit

범용 오류 타입 모음.

## Requirements

- macOS 12.4
- Xcode 13.4
- Swift 5.6

## Installation

### Swift Package Manager

```swift
.package(url: "https://github.com/jaemyeong/ErrorKit.git", .upToNextMajor(from: "0.1.7"))
```

## Usage

```swift
import ErrorKit

throw UnknownError()
```
