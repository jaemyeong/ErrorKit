# ErrorKit

일반적으로 사용 할 수 있는 전역 오류 타입 패키지.

## Requirements

- Xcode 13.2.1
- Swift 5.5

## Installation

### Swift Package Manager

```swift
.package(url: "https://github.com/jaemyeong/OpenColorKit.git", .upToNextMajor(from: "0.1.1"))
```

## Usage

```swift
import ErrorKit

throw UnknownError()
```
